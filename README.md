# AI Voice Negotiator 🎯

**Let AI negotiate better deals for you**

An intelligent web application that helps users negotiate better prices on subscriptions, bills, and purchases using AI-powered negotiation strategies.

## Features

- 📊 **Real-time Savings Tracker** - Monitor monthly and annual savings
- 🤝 **AI Negotiation Engine** - Automated negotiation simulation
- 💰 **Multiple Service Types** - Support for subscriptions, bills, insurance, and more
- 📱 **Responsive Design** - Works seamlessly on desktop and mobile
- 🎨 **Modern UI** - Beautiful gradient design with smooth animations

## Built With

- React 18 + TypeScript
- Vite for blazing fast development
- Modern CSS with glassmorphism effects
- Gradient color schemes and smooth transitions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## How It Works

1. **Add Negotiation** - Select the type of service, enter current and target prices
2. **AI Processes** - The AI analyzes market rates and negotiation strategies
3. **Track Results** - Monitor success rates and calculate total savings
4. **Save Money** - Enjoy lower bills and subscription costs

## Supported Negotiation Types

- Subscription Services (Netflix, Spotify, etc.)
- Internet/Cable Bills
- Phone Bills
- Insurance Premiums
- Gym Memberships
- Rent
- Other recurring expenses

## Trending in 2026
